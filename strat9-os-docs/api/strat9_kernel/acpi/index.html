Module acpi 

Source
Expand description

ACPI (Advanced Configuration and Power Interface) support. Inspired by Theseus OS, MaestroOS, Aero, and Redox.

Features:

  • RSDP/RSDT/XSDT parsing
  • MADT (interrupts, APICs)
  • FADT (power management, DSDT)
  • HPET (timers)
  • MCFG (PCIe MMCONFIG)
  • DMAR (IOMMU)
  • WAET (VM optimization hints)
  • BGRT (boot graphics)
  • SLIT (NUMA distances)
