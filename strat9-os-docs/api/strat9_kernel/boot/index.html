Boot subsystem

Regroups all code involved in the early kernel startup:

  • assembly stubs (16-bit → 64-bit transition)
  • bootloader handoff structures (KernelArgs)
  • Limine boot-protocol entry point
  • early serial logger
  • kernel panic handler
