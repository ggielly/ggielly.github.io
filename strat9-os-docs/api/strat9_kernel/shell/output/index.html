strat9_kernel::shell

Module output 

Source
Expand description

Shell output formatting and capture.

When capture mode is active, shell_print! / shell_println! write to an internal buffer instead of serial + VGA. This powers pipe (|) and redirection (>, >>) in the Chevron shell.
